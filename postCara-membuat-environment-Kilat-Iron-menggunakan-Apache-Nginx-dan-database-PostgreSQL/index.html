Membuat Environment Kilat Iron menggunakan Apache/Nginx dan database PostgreSQL

Author: Amar Rafli Date: December 2, 2019  16:52:48

Halo,

Pada kesempatan kali ini saya akan menjelaskan langkah - langkah untuk membuat Environment Kilat Iron menggunakan Apache/Nginx dan database PostgreSQL.

Berikut langkah - langkahnya :

  • Klik New Environment yang ada diatas (menu sejajar dengan logo Kilat Iron )

  • Lalu isikan owner dari Environment Anda.

  • Lalu terapkan Reserved untuk Cloudlet, Scalling limit, dll, berikut settingan untuk node Apache dan Nginx
  • Apache

  • Nginx

  • Lalu berikut settingan untuk node PostgreSQL

Penjelasan untuk beberapa point pada screenshot diatas :

  1. Reserved : Informasi CloudLet yang ditetapkan untuk node Anda.

    Scalling limit : Informasi CloudLet yang ditetapkan untuk auto scalling, CloudLet scalling limit akan digunakan apabila CloudLet reserved telah dipergunakan semuanya. Dimana satu CloudLet sama dengan 128 MB RAM dan 400Mhz Prosesor.

  2. Stateful dan Stateless digunakan untuk horizontal scalling, dimana fungsi nya untuk menambahkan node baru dengan template yang sama dengan master node (node utama).

    • Stateful : Membuat Node dengan isi dan template yang sama dengan master node nya (aplikasi yang disebelah kiri)

    • Stateless : Membuat node berdasarkan template master node nya (pembuatan node tidak beserta isi dari master node)

    • PHP 7.3.XX pada web server : Versi PHP yang dapat dipilih sesuai kebutuhan Anda

    • Disk Limit : Disk yang digunakan pada node aplikasi Anda

    • Sequential restart delay : Lama waktu delay setelah node diatas nya dilakukan restart/berurutan

    • IPV4 Public : Menu agar node Anda mendapatkan IP Public

  3. Environment Name : Nama Environment Anda

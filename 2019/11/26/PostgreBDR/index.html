PostgreBDR

Halo,

Saat ini saya akan menjelaskan tentang PostgresBDR.

1. Apa Itu PostgresBDR ?

PostgresBDR adalah sebuah aplikasi dari PostgreSQL, yang mana berfungsi sebagai replikasi database server. Spesifik nya replikasi database server master - master

Apa itu replikasi database ?, replikasi database adalah suatu sistem dimana fungsi nya sebagai backup dari database utama, ketika database utama mengalami error maka query yang masuk akan diproses oleh database yang lain.

Berikut cara instalasi PostgresBDR di sistem operasi Ubuntu 18.04 :

  • Menginstall Postgres-BDR melalui apt

Kalau tidak bisa melalui tutorial diatas, silakan untuk mencoba menginstall postgres-bdr nya melalui link berikut : https://www.2ndquadrant.com/en/resources/bdr/bdr-installation-instructions/

  • Buat directory postgresBDR, dan setting path serta inisiasi db
  • Edit file konfigurasi $HOME/9.4-bdr/postgresql.conf, menjadi berikut :
  • Lalu selanjutnya edit file konfigurasi $HOME/9.4-bdr/pg_hba.conf, menjadi berikut :
  • Setelah itu, silakan untuk menonaktifkan service PostgreSQL Anda lalu aktifkan service Postgres-BDR Anda. Setelah Postgres-BDR Anda aktif, silakan membuat user untuk Master dan set privileges nya menjadi superuser.
  • Create user biasa dan create database, serta set owner database Anda. Lalu tambahkan ekstensi bdr pada database Anda.
  • Setelah konfigurasi database master 1 dan database master 2, silakan membuat grup replikasi pada master 1, berikut :
  • Apabila Anda sudah membuat grup replikasi database pada master 1. Anda dapat membuat master 2 join ke grup master 1.
Note: Anda tidak perlu membuat grup pada master 2, Anda hanya perlu membuat master 2 join ke grup master 1.
